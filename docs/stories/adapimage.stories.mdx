## 基础属性

| 参数 | 说明          | 类型   | 可选值               | 默认值  |
| ---- | ------------- | ------ | -------------------- | ------- |
| ele  | 元素 dom 节点 | String | Class / id / TagName | TagName |

## 配置 options

| 参数         | 说明             | 类型   | 可选值 | 默认值 |
| ------------ | ---------------- | ------ | ------ | ------ |
| noTransition | 是否没有过渡效果 | Boolean | --     | false   |
