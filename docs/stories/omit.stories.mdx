# 响应式文本截断（Omit）

---

<Meta title="常用函数/响应式文本截断(Omit)" />

<Canvas>
    <Story
        name="响应式文本截断(Omit)"
        height="150px"
        parameters={{
            docs: { source: { code: `${viewinfo.html}${viewinfo.js}` } },
        }}
    >
        {() => {
            useEffect(() => {
                Function(viewinfo.js)();
            });
            return viewinfo.html;
        }}
    </Story>
</Canvas>

## 基础属性

| 参数 | 说明          | 类型   | 可选值               | 默认值  |
| ---- | ------------- | ------ | -------------------- | ------- |
| ele  | 元素 dom 节点 | String | Class / id / TagName | TagName |

## 配置 options

| 参数       | 说明                             | 类型    | 可选值 | 默认值 |
| ---------- | -------------------------------- | ------- | ------ | ------ |
| breakpoint | 断点                             | Object  |   --   |   --   |
| isTwoway   | 双向（分辨率低到高或高到低）切换 | Boolean | --     | false     |


## 回调方法 on

| 参数   | 说明       | 类型     | 默认值 | 回调参数 |
| ------ | ---------- | -------- | ------ | -------- |
| resize | 初始化回调 | Function | null   | event↓   |

| event 参数 | 说明            | 类型   |
| ---------- | --------------- | ------ |
| ele        | 文本域 DOM 集合 | Array  | Object |
| eventName  | 当前回调事件名  | String |
