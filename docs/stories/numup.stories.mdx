## 基础属性

| 参数 | 说明          | 类型   | 可选值               | 默认值  |
| ---- | ------------- | ------ | -------------------- | ------- |
| ele  | 元素 dom 节点 | String | Class / id / TagName | TagName |

## 配置 options

如果标签上含有 hope-numup-\*\*相应属性可以不写 options 配置

| 参数  | 说明         | 类型   | 可选值 | 默认值 |
| ----- | ------------ | ------ | ------ | ------ |
| num   | 要显示的数字 | Number | --     | --     |
| time  | 效果总耗时   | Number | --     | 2000   |
| speed | 动画速度     | Number | --     |        | 100 |
| start | 开始数字     | Number | --     |        | 0 |

## 回调方法 on

| 参数 | 说明   | 类型     | 默认值 | 回调参数 |
| ---- | ------ | -------- | ------ | -------- |
| end  | 完成后 | Function | null   | event    |
