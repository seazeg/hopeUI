## 基础属性

| 参数 | 说明          | 类型   | 可选值               | 默认值  |
| ---- | ------------- | ------ | -------------------- | ------- |
| ele  | 元素 dom 节点 | String | Class / id / TagName | TagName |

## 配置 options

| 参数       | 说明 | 类型   | 可选值 | 默认值 |
| ---------- | ---- | ------ | ------ | ------ |
| breakpoint | 断点 | Object | --     | --     |

## 回调方法 on

| 参数   | 说明       | 类型     | 默认值 | 回调参数 |
| ------ | ---------- | -------- | ------ | -------- |
| resize | 初始化回调 | Function | null   | event↓   |

| event 参数 | 说明           | 类型   |
| ---------- | -------------- | ------ |
| ele        | 图片 DOM 集合  | Array  | Object |
| eventName  | 当前回调事件名 | String |
