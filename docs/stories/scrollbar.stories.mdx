## 基础属性

| 参数 | 说明          | 类型   | 可选值               | 默认值  |
| ---- | ------------- | ------ | -------------------- | ------- |
| ele  | 元素 dom 节点 | String | Class / id / TagName | TagName |


## 配置 options

| 参数 | 说明          | 类型   | 可选值               | 默认值  |
| ---- | ------------- | ------ | -------------------- | ------- |
| height  | 内容高度 | Number | -- | 200 |



## 回调方法 on

| 参数  | 说明         | 类型     | 默认值 | 回调参数     |
| ----- | ------------ | -------- | ------ | ------------ |
| init  | 初始化回调   | Function | null   | 当前对象 DOM |
| scroll  | 滚动的位移值 | Function | null   | event↓       |


| event 参数 | 说明           | 类型   |
| ---------- | -------------- | ------ |
| distance  | 位移值   | Number |
| eventName  | 当前回调事件名 | String |

## 调用方法

| 方法名 | 说明     | 类型     | 默认值 | 参数 |
| ------ | -------- | -------- | ------ | ---- |
| val    | 外部赋值 | Function | --     | --   |
| clear  | 外部清值 | Function | --     | --   |
